# Qwen Text Failure Report Page

Editable page:

```text
research_pages/qwen_text_failure/index.html
```

Serve from the repository root:

```bash
python -m http.server 8090 --bind 127.0.0.1
```

Open:

```text
http://127.0.0.1:8090/research_pages/qwen_text_failure/
```

The report images and CSVs needed for reading the page are copied into
`research_pages/qwen_text_failure/assets/`, so the GitHub version does not
depend on the local `outputs` symlink.
