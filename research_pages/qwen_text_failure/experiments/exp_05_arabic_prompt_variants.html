Experiment 05: Arabic Prompt Variants

Tests whether prompt wording alone can repair Arabic text fidelity.

Observed Result

Quoted text, Unicode wording, explicit RTL instruction, an Arabic-language prompt, and word-level spelling instructions all fail to recover the target phrase. Prompt phrasing alone is therefore not a reliable fix for this Arabic case.
